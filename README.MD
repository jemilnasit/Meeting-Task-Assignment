# Meeting Task Assignment

#### Project file structure
- main.py
- model_training.py
- speech_to_text.py
- task_classifier.py
- requirements.txt

#### -> main.py file:
- Assigned task(cosine_similarity)
- Deadline and priority (rule based)

#### -> model_training.py file:
- model training for task classification
- use sentence transformer for convert text into embaddings

#### -> speech_to_text.py file:
- take a audio as input and convert to text using whisper

#### -> task_classifier.py file:
- preprocess text and find out task summary
- classify task using trained model

#### -> requirements.txt file:
- it contains dependencies or packages that is required in project
